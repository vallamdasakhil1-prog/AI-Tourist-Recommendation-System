# 🌍 AI Tourist Recommendation System

An AI-powered tourist recommendation system that suggests nearby tourist places using Machine Learning and real-time location data.

## 🚀 Features
- 📍 GPS-based location detection
- 🗺 Interactive map selection
- 🤖 ML-based tourist ranking
- 🌐 OpenTripMap API integration
- 🏆 Top recommended tourist places
- 📌 Category classification
- 🧭 Google Maps directions
- 🎨 Streamlit interactive UI
- ⚡ Flask lightweight web interface

---

## 🛠 Technologies Used
- Python
- Streamlit
- Flask
- Scikit-learn
- Random Forest Regression
- Folium
- OpenTripMap API
- Geopy
